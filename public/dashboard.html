Getting Started with GRASP

Follow this step-by-step guide to create AI-powered questions from your course materials and manage them effectively.

1

Upload Your Course Materials

Start by uploading your course documents (PDFs, lecture notes, presentations, text files) through the Upload quick start card or Course Materials page. These documents will be processed and stored for question generation.

Supported formats: PDF, DOC, DOCX, TXT. You can also paste text directly or add URLs.
2

Navigate to Question Generation

Navigate to the Question Generation page. This is where you'll create learning objectives and generate questions from your uploaded materials.

3

Create Learning Objectives

Create learning objectives (LOs) for your course. Each objective can have multiple granular objectives that break down the main objective into smaller, specific learning goals.

  • Create custom objectives manually, or use AI to generate them from your content(Coming soon)
  • Add granular objectives to each main learning objective
  • Select Bloom's Taxonomy levels (Remember, Understand, Apply, Analyze, Evaluate, Create) for each granular objective
  • Each learning objective must have at least 5 questions total (across all its granular objectives)
4

Attach Documents to Learning Objectives

For each learning objective, attach the relevant uploaded documents. This ensures the AI uses the correct information when generating questions specific to that objective.

The AI will only use content from attached documents, ensuring questions are relevant and accurate.
5

Generate Questions with AI

Click Generate Questions to create multiple-choice questions automatically. The AI analyzes the attached documents and generates questions aligned with your learning objectives and selected Bloom's levels.

  • Review and edit generated questions before saving
  • Regenerate individual questions or all questions at once
  • Edit question text, options, and correct answers
  • Flag questions that need attention
6

Save Questions to Question Bank

Click Add to Question Bank to save your questions. You'll be prompted to either:

  • Create a new quiz: Enter a quiz name and description, then save
  • Add to existing quiz: Select from your existing quizzes

Questions are automatically linked to the selected quiz and saved to your Question Bank.

Question Bank: Your Central Hub

The Question Bank is where you manage all your questions and quizzes. It has three main tabs:

Overview Tab

View and manage all questions across all quizzes:

  • View/Edit: Click the button that appears on hover to open a detailed modal where you can edit question title, stem, options, and correct answer
  • Approve/Unapprove: Change question status to control which questions are ready for use
  • Filter: Narrow down questions by Quiz, Learning Objective, Bloom Level, Status (Approved/Draft), or Flagged status
  • Search: Find questions by keywords in titles or content
  • Sort: Organize questions by title, learning objective, Bloom level, or status
  • Bulk Actions: Select multiple questions to flag or delete them at once
  • Flag Questions: Mark questions that need review or attention
Review Tab

Review questions organized by quiz:

  • See review progress percentage for each quiz
  • View all questions within each quiz
  • Approve or flag questions directly from quiz view
  • Click Review button to jump to Overview tab with that quiz filtered and status set to Draft
Approved History Tab

Track your approved questions over time (coming soon)

Pro Tips

  • Use granular objectives to create more specific, targeted questions
  • Regenerate questions if the initial results don't meet your needs
  • Use Bloom's Taxonomy levels strategically to create questions at different cognitive levels
  • Flag questions during generation to review them later in the Question Bank
  • Organize questions into multiple quizzes for different topics or assessments
  • Approve questions only after thorough review to maintain quality
